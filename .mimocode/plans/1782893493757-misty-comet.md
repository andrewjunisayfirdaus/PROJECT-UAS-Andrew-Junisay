# Plan: Alarm Suara Pengingat Kantuk pada Face Monitor

## Context
Halaman Face Monitor mendeteksi kantuk pengemudi via webcam (blink count, EAR, yawn count), namun hanya menampilkan status secara visual (Badge warna). Tidak ada alarm audio yang berbunyi saat pengemudi mulai mengantuk. Pengguna meminta alarm suara yang berbunyi saat status berubah menjadi "Drowsy".

## Approach
Tambahkan alarm suara menggunakan **Web Audio API** (tanpa file audio eksternal). Alarm berbunyi sebagai beep pattern saat status driver berubah menjadi "Drowsy" atau "Critical", dan berhenti otomatis saat status kembali "Alert".

## Files to Modify
1. **`app/detection/face-monitor/page.tsx`** — tambah alarm logic + UI toggle

## Implementation Details

### 1. Buat fungsi generate beep menggunakan Web Audio API
- Fungsi `playAlarmSound()` menggunakan `AudioContext` + `OscillatorNode`
- Generate beep pattern: 3 kali beep (frekuensi ~800ms, interval 200ms)
- Frekuensi ~800Hz, durasi ~300ms per beep, dengan pause antar beep

### 2. State baru untuk alarm control
- `isAlarmEnabled: boolean` (default `true`) — toggle aktif/nonaktif alarm
- `alarmPlayed: boolean` — flag untuk mencegah alarm berulang terus-menerus
  - Reset ke `false` saat status kembali "Alert"

### 3. Logic trigger alarm
- Di `useEffect` yang sudah ada (dependency `blinkCount`, `yawnCount`):
  - Saat status berubah ke "Drowsy" atau "Critical" → panggil `playAlarmSound()` jika `isAlarmEnabled && !alarmPlayed`
  - Saat status berubah ke "Alert" → reset `alarmPlayed = false`
- Alarm hanya berbunyi **satu kali** per transisi status (tidak berulang terus)

### 4. UI Toggle Alarm
- Tambahkan tombol toggle alarm di panel "Driver Status Panel" (sebelah kanan)
- Menggunakan komponen `Switch` yang sudah ada di `components/ui/switch.tsx`
- Label: "Alarm Pengingat" dengan status ON/OFF
- Ikon: `IconBell` / `IconBellOff` dari `@tabler/icons-react`

## Verification
1. Jalankan app dengan `npm run dev`
2. Buka halaman `/detection/face-monitor`
3. Nyalakan kamera → deteksi wajah aktif
4. Klik tombol "Simulate Yawn" beberapa kali sampai yawnCount > 3
5. **Verifikasi:** Alarm suara berbunyi (3 kali beep)
6. Klik tombol "Reset Blinks" → yawnCount reset
7. **Verifikasi:** Alarm berhenti berbunyi, tidak ada suara lagi
8. Toggle alarm OFF → ulangi langkah 4
9. **Verifikasi:** Tidak ada suara alarm meskipun status "Drowsy"
