# Rencana: Koneksi Smartwatch untuk Deteksi Detak Jantung

## Ringkasan
Menambahkan fitur koneksi Bluetooth (Web Bluetooth API) ke halaman Health agar dapat menerima data detak jantung secara real-time dari smartwatch (Xiaomi Mi Band, Huawei Band, Fitbit, dll).

## Pendekatan: Web Bluetooth API
- API browser bawaan (tidak perlu install package tambahan)
- Mendukung BLE (Bluetooth Low Energy) devices
- Heart Rate Service UUID: `0x180D`, Characteristic UUID: `0x2A37`
- Kompatibel: Chrome, Edge (tidak support Firefox/Safari)
- Fallback ke dummy data jika browser tidak support atau device tidak tersambung

## File yang akan dibuat/diubah

### 1. `hooks/use-bluetooth-heart-rate.ts` (BARU)
Custom hook untuk manajemen koneksi BLE:
- `connect()` - scan & connect device
- `disconnect()` - putuskan koneksi
- `heartRate` - nilai detak jantung terkini
- `heartRateHistory` - array history detak jantung (24 data point terakhir)
- `isConnected` - status koneksi
- `isSupported` - apakah browser support Web Bluetooth
- `deviceName` - nama device yang terhubung
- `batteryLevel` - level baterai device
- `error` - pesan error jika ada
- Subscribe ke heart rate notifications via `characteristic.startNotifications()`

### 2. `context/bluetooth-provider.tsx` (BARU)
Context provider untuk state Bluetooth global:
- Wrap di `app/layout.tsx` agar bisa diakses dari halaman manapun
- Menyediakan `useBluetooth()` hook
- Mengelola connection state global

### 3. `components/shared/bluetooth-connection.tsx` (BARU)
UI komponen koneksi Bluetooth:
- Tombol "Hubungkan Smartwatch" (ikon bluetooth)
- Status indicator (terhubung/terputus/not supported)
- Info device: nama, baterai
- Tombol disconnect
- Loading state saat scanning
- Error toast jika gagal

### 4. `app/history/health/page.tsx` (UBAH)
Modifikasi halaman health:
- Import `useBluetooth` hook
- Tampilkan `BluetoothConnection` component di bagian atas
- Heart Rate card menampilkan data real-time jika terhubung, dummy jika tidak
- Heart Rate chart menampilkan history real-time dari bluetooth
- Auto-update setiap kali ada notifikasi baru

### 5. `lib/types.ts` (UBAH)
Tambahkan type:
```ts
interface BluetoothDevice {
  name: string;
  heartRate: number;
  batteryLevel: number;
}
```

### 6. `app/layout.tsx` (UBAH)
- Wrap children dengan `BluetoothProvider`

## Alur Kerja

```
User klik "Hubungkan Smartwatch"
    ↓
Web Bluetooth API: navigator.bluetooth.requestDevice()
    ↓ (device filter: Heart Rate Service 0x180D)
Device picker muncul → user pilih device
    ↓
Connect ke GATT server
    ↓
Get Heart Rate Service (0x180D)
    ↓
Get Heart Rate Measurement Characteristic (0x2A37)
    ↓
Start notifications → listen for heart rate data
    ↓
Parse heart rate value → update UI real-time
```

## Error Handling
- Browser tidak support → tampilkan pesan "Browser tidak mendukung Bluetooth"
- User cancel device picker → handle AbortError gracefully
- Device disconnect → auto-reconnect attempt + status update
- Gagal connect → toast error dengan pesan yang jelas
